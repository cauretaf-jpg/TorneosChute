# Chute Plataforma 1.8.3 · Perfiles de futbolistas

Esta versión agrega fichas competitivas para los futbolistas oficiales de Chute.

## Incluye

- Perfil individual de futbolista.
- Foto, equipo y posición oficial.
- Goles, asistencias y G+A históricos.
- Ranking global y ranking dentro del club.
- Usuarios que más han registrado acciones con el futbolista.
- Torneos destacados.
- Últimos registros de goles/asistencias.
- Selección desde Ranking, Fichas y perfiles de equipo.

## Base de datos

No requiere SQL nuevo. Usa los eventos de goles y asistencias ya guardados en los partidos.
