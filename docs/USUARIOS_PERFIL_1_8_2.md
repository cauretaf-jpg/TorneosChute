# Chute Plataforma 1.8.2 · Perfiles de usuario

Esta versión agrega perfiles competitivos de usuario con una presentación similar a los perfiles de equipo.

## Cambios principales

- En Ranking > Usuarios se puede seleccionar un usuario y abrir su ficha competitiva.
- La pestaña Perfil del usuario conectado usa el nuevo diseño completo.
- La pestaña Fichas incorpora el perfil completo de usuario junto a la ficha de equipo.
- Cada perfil de usuario muestra:
  - ranking global;
  - partidos, victorias, empates y derrotas;
  - goles a favor, goles en contra y diferencia;
  - títulos, rendimiento y score;
  - torneos jugados y finalizados;
  - subcampeonatos;
  - equipo más usado;
  - mejor equipo;
  - rival frecuente;
  - mejores equipos usados;
  - últimos partidos;
  - rivales frecuentes;
  - futbolistas destacados;
  - palmarés;
  - logros.

## Supabase

No requiere SQL nuevo. Usa los datos ya cargados en torneos, partidos, rankings, eventos de goles/asistencias e historial.
