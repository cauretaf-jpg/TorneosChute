# Chute Plataforma 1.2 - Amigos reales

Esta versión conecta el módulo de amigos con Supabase.

## Qué queda conectado

- Búsqueda de usuarios registrados desde `profiles`.
- Solicitudes de amistad en `friendships`.
- Aceptar o rechazar solicitudes recibidas.
- Lista de amigos aceptados vinculada al usuario autenticado.

## Qué sigue local todavía

- Creación de torneos.
- Invitaciones a torneos.
- Participantes.
- Partidos.
- Goles y asistencias.
- Rankings históricos completos.

## SQL recomendado

Ejecutar `supabase/06_friendships_1_2.sql` después de `05_auth_profile_policy.sql`.
