# Chute Plataforma 1.8.4 · Perfil de torneos

Esta versión agrega una ficha competitiva para cada torneo.

## Cambios

- Nueva pestaña **Perfil** dentro de la sala del torneo.
- Ficha tipo álbum histórico con campeón/líder, formato, modo de equipos y temporada.
- Resumen visual de participantes, partidos, goles, goleador, asistidor, mejor futbolista, mejor ataque y mejor defensa.
- Bloque de podio y rendimiento.
- Tabla final/resumen de posiciones dentro del perfil.
- Partidos destacados ordenados por cantidad de goles y diferencia.
- Álbum histórico de torneos finalizados en Mundo Chute y Ranking > Palmarés.

## Base de datos

No requiere SQL nuevo. Usa los torneos, partidos, goles, asistencias y resúmenes históricos ya existentes.
